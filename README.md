## my neovim config

this is a simple neovim config from [Lazyvim](https://github.com/LazyVim/LazyVim) with some additional plugins (mostly from [folke](https://github.com/folke))

to download this config you need:

[neovim](https://github.com/neovim/neovim/blob/master/INSTALL.md)

and this command:

```bash
git clone https://github.com/amanafro/nvim ~/.config/nvim/
```

enjoy😃
